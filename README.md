# brick-breaker-game-java
# 🧱 Brick Breaker Game (Java Swing)

## 📌 Project Description
This is a simple Brick Breaker game built using Java Swing.  
The game includes paddle control, ball physics, brick collision, and scoring system.

---

## 🎮 Features
- Start Menu Screen
- Paddle movement using keyboard
- Ball collision detection
- Brick destruction system
- Score tracking
- Game Over detection

---

## 🧩 Modules
- Main Class → Starts the game
- MenuPanel → Start screen UI
- GamePanel → Game logic and rendering
- MapGenerator → Creates brick layout

---

## ⚙️ Technologies Used
- Java
- Swing (GUI)
- AWT

---

## ▶️ How to Run
1. Clone the repository
2. Open project in IDE (Eclipse / IntelliJ / NetBeans)
3. Run `Main.java`
